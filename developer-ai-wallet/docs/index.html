SDK

Use the SDK inside your own agent service. The wallet key stays with the developer runtime.

import { TitanAgentWalletClient } from "@titan/agent-wallet";

const client = new TitanAgentWalletClient({
  baseUrl: "https://wallet.yieldboostai.xyz/api",
  militaryBaseUrl: "https://wallet.yieldboostai.xyz",
  projectId: process.env.TITAN_AGENT_WALLET_PROJECT_ID,
  agentWalletId: process.env.TITAN_AGENT_WALLET_ID,
  capabilityToken: process.env.TITAN_AGENT_WALLET_CAPABILITY,
});

await client.sendNative({
  privateKey: process.env.TITAN_AGENT_WALLET_PRIVATE_KEY,
  rpcUrl: "https://evmrpc.0g.ai",
  chainId: 16661,
  networkName: "0G Mainnet",
  to: "0x...",
  valueEth: "0.001",
  context: { intent: "Pay approved vendor invoice" },
});

CLI

The CLI mirrors the SDK for local agents, scripts, and CI.

npm run build
node dist/cli.js health
node dist/cli.js check-intent \
  --intent "Pay approved vendor invoice"
node dist/cli.js send \
  --private-key "$TITAN_AGENT_WALLET_PRIVATE_KEY" \
  --rpc-url "https://evmrpc.0g.ai" \
  --to "0x..." \
  --value-eth "0.001" \
  --intent "Pay approved vendor invoice"
